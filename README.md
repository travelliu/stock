# Stock Daily — A股日线价差分析工具

基于 Tushare API 的中国 A 股日线数据获取与价差统计分析 CLI 工具。

## 功能

- **数据获取**: 从 Tushare 拉取日线 OHLCV 数据，支持增量更新
- **价差计算**: 自动计算 6 种价差（高-开、开-低、高-低、开-收、高-收、低-收）
- **统计分析**: 多时间窗口（历史/近3月/近1月/近2周）的均值、中位数、众数
- **分布展示**: 直方图式分布表，多个时间窗口并排对比
- **交易计划**: 传入当日开盘价，生成基于历史价差的参考价预测表
- **实时反推**: 传入当日最低/最高/收盘价，反推收盘价的预测区间
- **终端友好**: CJK 字符对齐，中文表头正确显示

## 快速开始

### 1. 安装依赖

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 2. 配置 Tushare Token

复制 `.env.example` 为 `.env`，填入你的 Tushare token：

```bash
cp .env.example .env
```

```env
TUSHARE_TOKEN=你的tushare_token
```

Token 申请地址: [https://tushare.pro/register](https://tushare.pro/register)

### 3. 获取数据

```bash
# 增量获取（从上次记录日期开始）
python stock.py fetch --stocks 603778,000890

# 指定天数
python stock.py fetch --stocks 603778 --days 90

# 全量历史（回溯约10年）
python stock.py fetch --stocks 603778 --days all
```

### 4. 查看分析

```bash
# 默认显示近30日数据 + 价差分析
python stock.py show --stock 603778

# 指定日期范围
python stock.py show --stock 603778 --from 2025-04-01 --to 2025-05-12

# 显示全部6种价差
python stock.py show --stock 603778 --all
```

### 5. 交易计划

传入当日实时价格，生成基于历史价差的参考价预测：

```bash
# 基础交易计划（传入开盘价）
python stock.py show --stock 603778 --open 13.50

# 传入更多实时价格（反推收盘价区间）
python stock.py show --stock 603778 --open 13.50 --low 13.20 --high 13.80

# 传入收盘价（header 显示实际收盘价而非预测值）
python stock.py show --stock 603778 --open 13.50 --low 13.20 --high 13.80 --close 13.60
```

## 价差说明

所有价差均以**绝对值**存储，即 `|A − B|`。

| 价差              | 计算                | 用途                                |
|-------------------|---------------------|-------------------------------------|
| 高-开 (spread_oh) | \|最高价 − 开盘价\| | 高抛目标参考：开盘价 + 此值附近卖出 |
| 开-低 (spread_ol) | \|开盘价 − 最低价\| | 低吸目标参考：开盘价 − 此值附近买入 |
| 高-低 (spread_hl) | \|最高价 − 最低价\| | 全日振幅，衡量做T空间是否充足       |
| 开-收 (spread_oc) | \|开盘价 − 收盘价\| | 日内偏离幅度，衡量多空分歧程度      |
| 高-收 (spread_hc) | \|最高价 − 收盘价\| | 盘中高点到收盘的回落空间            |
| 低-收 (spread_lc) | \|最低价 − 收盘价\| | 盘中低点到收盘的反弹空间            |

默认只显示高-开和开-低两种（做T核心指标），使用 `--all` 查看全部。

## 交易计划输出

当传入 `--open` 时，输出会额外增加**交易计划**部分，包含：

### 1. 价差模型表

展示 6 种价差在 4 个时间窗口的均值：

```shell
+--------+------------+------------+------------+------------+------------+------------+
| 时段   | 开盘与最高价 | 开盘与最低价 | 最高价与最低价 | 开盘与收盘价 | 最高价与收盘价 | 最低价与收盘价 |
+--------+------------+------------+------------+------------+------------+------------+
| 历史   | 1.00       | 0.50       | 1.50       | 0.50       | 0.50       | 1.00       |
| 近3月  | 0.50       | 0.25       | 0.75       | 0.25       | 0.25       | 0.50       |
| 近1月  | 0.30       | 0.15       | 0.45       | 0.15       | 0.15       | 0.30       |
| 近2周  | 0.20       | 0.10       | 0.30       | 0.10       | 0.10       | 0.20       |
| 综合均值 | 0.50       | 0.25       | 0.75       | 0.25       | 0.25       | 0.50       |
+--------+------------+------------+------------+------------+------------+------------+
```

### 2. 历史参考价表

基于开盘价和各时段价差均值，预测当日最高/最低/收盘价：

```shell
+------------+------------+-------------+-------------+-------------+------------------------+------------------------+--------+----------+
|            | 历史参考价 | 近3月参考价 | 近1月参考价 | 近2周参考价 | 最低价反推(当日最低价) | 最高价反推(当日最高价) | 均值   | 正负算一 |
+------------+------------+-------------+-------------+-------------+------------------------+------------------------+--------+----------+
| 最高价预测 | 203.77     | 202.41      | 201.72      | 202.22      | /                      | /                      | 202.53 | +        |
| 最低价预测 | 196.18     | 197.50      | 198.41      | 198.56      | /                      | /                      | 197.66 | -        |
| 收盘价预测 | /          | /           | /           | /           | /                      | /                      | /      | -        |
+------------+------------+-------------+-------------+-------------+------------------------+------------------------+--------+----------+
```

**各列含义**:

| 列名                   | 含义                                              |
|------------------------|---------------------------------------------------|
| 历史/近3月/近1月/近2周 | 基于对应时段价差均值 + 开盘价计算预测价格          |
| 最低价反推(当日最低价) | `actual_low + 近2周 spread_hl`（反推最高价）      |
| 最高价反推(当日最高价) | `actual_high - 近2周 spread_hl`（反推最低价）     |
| 均值                   | 参考价列的平均值                                  |
| 正负算一               | `+` 表示该行为上限参考，`-` 表示下限参考          |

当传入 `--low`/`--high` 时，收盘价预测行的反推列会显示：
- **最低价反推** = `actual_low + 近2周 spread_lc`（从最低价反推收盘价）
- **最高价反推** = `actual_high - 近2周 spread_hc`（从最高价反推收盘价）
- **均值** = 两个反推值的平均数

## 统计输出说明

### 汇总表

每种价差在 4 个时间窗口分别统计：

| 时段   | 含义               |
|--------|--------------------|
| 历史   | 所有历史数据       |
| 近3月  | 最近约 3 个月      |
| 近1月  | 最近约 1 个月      |
| 近2周  | 最近约 2 周        |

统计指标：

| 指标   | 含义                                             |
|--------|--------------------------------------------------|
| 样本数 | 该时段内的交易日数量                             |
| 均值   | 所有价差的算术平均                               |
| 中位数 | 价差排序后的中间值（不受极端值影响，参考性更强） |
| 众数   | 出现频率最高的价差值（反映最常见的波动幅度）     |

高-开和开-低的汇总表**并排显示**，方便对比高抛和低吸空间。使用 `--all` 时，6 种价差按两两分组并排。

### 分布表

每种价差在 4 个时间窗口各生成一张分布表，展示价差落在各个区间的频率。

- 将该时段内所有价差值等分为 **10 个区间**（bin）
- 每个区间显示：区间范围、落在该区间的天数、占总天数的百分比
- 同一价差的 4 个时间窗口分布表并排对比

**如何解读**：

- 若前 2~3 个区间占比超过 60%，说明价差高度集中，目标价位可信度高
- 若分布平坦或尾部厚重，说明波动不可预测，做T目标应放宽
- 对比近2周与历史数据的分布，可判断近期波动是否偏离历史常态

**解读**:

- 对比两个窗口可判断：当前处于异常高波动期，做T空间充足
