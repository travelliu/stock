# Stock Daily — A股日线价差分析工具

基于 Tushare API 的中国 A 股日线数据获取与价差统计分析 CLI 工具。

## 功能

- **数据获取**: 从 Tushare 拉取日线 OHLCV 数据，支持增量更新
- **价差计算**: 自动计算 6 种价差（高-开、开-低、高-低、开-收、高-收、低-收）
- **统计分析**: 多时间窗口（全部/近90日/近30日/近15日）的均值、中位数、众数
- **分布展示**: 直方图式分布表，多个时间窗口并排对比
- **终端友好**: CJK 字符对齐，中文表头正确显示

## 快速开始

### 1. 安装依赖

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 2. 配置 Tushare Token

复制 `.env.example` 为 `.env`，填入你的 Tushare token：

```bash
cp .env.example .env
```

```env
TUSHARE_TOKEN=你的tushare_token
```

Token 申请地址: [https://tushare.pro/register](https://tushare.pro/register)

### 3. 获取数据

```bash
# 增量获取（从上次记录日期开始）
python stock.py fetch --stocks 603778,000890

# 指定天数
python stock.py fetch --stocks 603778 --days 90

# 全量历史（回溯约10年）
python stock.py fetch --stocks 603778 --days all
```

### 4. 查看分析

```bash
# 默认显示近30日数据 + 价差分析
python stock.py show --stock 603778

# 指定日期范围
python stock.py show --stock 603778 --from 2025-04-01 --to 2025-05-12

# 显示全部6种价差
python stock.py show --stock 603778 --analyze --all
```

## 价差说明

所有价差均以**绝对值**存储，即 `|A − B|`。

| 价差              | 计算                | 用途                                |
|-------------------|---------------------|-------------------------------------|
| 高-开 (spread_oh) | \|最高价 − 开盘价\| | 高抛目标参考：开盘价 + 此值附近卖出 |
| 开-低 (spread_ol) | \|开盘价 − 最低价\| | 低吸目标参考：开盘价 − 此值附近买入 |
| 高-低 (spread_hl) | \|最高价 − 最低价\| | 全日振幅，衡量做T空间是否充足       |
| 开-收 (spread_oc) | \|开盘价 − 收盘价\| | 日内偏离幅度，衡量多空分歧程度      |
| 高-收 (spread_hc) | \|最高价 − 收盘价\| | 盘中高点到收盘的回落空间            |
| 低-收 (spread_lc) | \|最低价 − 收盘价\| | 盘中低点到收盘的反弹空间            |

默认只显示高-开和开-低两种（做T核心指标），使用 `--all` 查看全部。

## 统计输出说明

### 汇总表

每种价差在 4 个时间窗口分别统计：

| 时段   | 含义             |
|--------|------------------|
| 全部   | 所有历史数据     |
| 近90日 | 最近 90 个交易日 |
| 近30日 | 最近 30 个交易日 |
| 近15日 | 最近 15 个交易日 |

统计指标：

| 指标   | 含义                                             |
|--------|--------------------------------------------------|
| 样本数 | 该时段内的交易日数量                             |
| 均值   | 所有价差的算术平均                               |
| 中位数 | 价差排序后的中间值（不受极端值影响，参考性更强） |
| 众数   | 出现频率最高的价差值（反映最常见的波动幅度）     |

高-开和开-低的汇总表**并排显示**，方便对比高抛和低吸空间。使用 `--all` 时，6 种价差按两两分组并排。

### 分布表

每种价差在 4 个时间窗口各生成一张分布表，展示价差落在各个区间的频率。

- 将该时段内所有价差值等分为 **10 个区间**（bin）
- 每个区间显示：区间范围、落在该区间的天数、占总天数的百分比
- 同一价差的 4 个时间窗口分布表并排对比

**如何解读**：

- 若前 2~3 个区间占比超过 60%，说明价差高度集中，目标价位可信度高
- 若分布平坦或尾部厚重，说明波动不可预测，做T目标应放宽
- 对比近15日与全部数据的分布，可判断近期波动是否偏离历史常态

## 输出示例

以 `python stock.py show --stock xxxxxx` 为例，输出分为三部分：

### 1. 汇总表（并排对比高抛/低吸空间）

```shell
── 最高-开盘 汇总 ──                          ── 开盘-最低 汇总 ──
+--------+--------+------+--------+------+    +--------+--------+------+--------+------+
| 时段   | 样本数 | 均值 | 中位数 | 众数 |    | 时段   | 样本数 | 均值 | 中位数 | 众数 |
+--------+--------+------+--------+------+    +--------+--------+------+--------+------+
| 全部   | 2384   | 0.12 | 0.07   | 0.00 |    | 全部   | 2384   | 0.11 | 0.07   | 0.00 |
| 近90日 | 90     | 0.41 | 0.23   | 0.00 |    | 近90日 | 90     | 0.34 | 0.24   | 0.00 |
| 近30日 | 30     | 0.67 | 0.55   | 0.35 |    | 近30日 | 30     | 0.56 | 0.52   | 0.71 |
| 近15日 | 15     | 0.72 | 0.58   | 0.28 |    | 近15日 | 15     | 0.66 | 0.62   | 0.71 |
+--------+--------+------+--------+------+    +--------+--------+------+--------+------+
```

**解读**: 以近15日为例，开盘后大概率上涨 0.58 元（中位数），下跌 0.62 元（中位数）。上下空间接近，适合双向做T。

### 2. 分布表（4个时间窗口并排）

```shell
── 最高-开盘 分布 (全部,2384条) ──    ── 最高-开盘 分布 (近15日,15条) ──
+-----------+------+-------+          +-----------+------+-------+
| 区间      | 数量 | 占比  |          | 区间      | 数量 | 占比  |
+-----------+------+-------+          +-----------+------+-------+
| 0.00~0.17 | 1911 | 80.2% |          | 0.03~0.20 | 3    | 20.0% |
| 0.17~0.34 | 292  | 12.2% |          | 0.20~0.37 | 2    | 13.3% |
| 0.34~0.52 | 102  | 4.3%  |          | 0.37~0.54 | 1    | 6.7%  |
| ...       | ...  | ...   |          | ...       | ...  | ...   |
+-----------+------+-------+          +-----------+------+-------+
```

**解读**:

- 全部数据中 80.2% 集中在 0~0.17，历史上该股波动很小
- 近15日分布明显右移，集中在 0.3~1.4 区间，说明近期波动放大了数倍
- 对比两个窗口可判断：当前处于异常高波动期，做T空间充足
