## 项目概述
基于现有股票分析CLI工具，设计一个Web前端界面，提供股票数据可视化、分析和管理功能。

## 技术栈
参考 `/root/code/gitlab/mogdb_en/mtk/pkg/mtkd/web` 代码风格：
- **Vue 3** - 前端框架
- **Element Plus** - UI组件库
- **Vue Router** - 路由管理
- **Vue I18n** - 国际化
- **Axios** - HTTP请求
- **SCSS** - 样式预处理器
- **Vite** - 构建工具

## 详细页面设计

### 1. 股票列表页面 (`StockListView.vue`)
**功能**：
- 显示所有关注的股票列表
- 支持搜索、筛选、排序
- 显示股票基本信息（代码、名称、最新价格、涨跌幅）
- 点击进入详情页面

**UI设计**：
- 顶部：搜索框、筛选条件（市场、板块）
- 中部：股票列表表格
- 底部：分页组件

**数据字段**：
- 股票代码 (6位数字)
- 股票名称
- 最新价格
- 涨跌幅
- 成交量
- 最后更新日期

### 2. 股票详情页面 (`StockDetailView.vue`)
**功能**：
- 显示单个股票的详细信息
- 2个Tab页设计

**Tab 1: 基础信息与价差分析**
- 股票基本信息展示
- 6种价差展示（高-开、开-低、高-低、开-收、高-收、低-收）
- 近期日线数据表格
- 价差分布直方图

**Tab 2: 详细统计分析**
- 多时间窗口统计分析（历史/近3月/近1月/近2周）
- 价差模型表（均值、中位数、众数）
- 交易计划生成（基于开盘价预测）
- 实时反推功能（基于最低/最高价反推收盘价）

**UI设计**：
- 顶部：股票基本信息卡片
- 中部：Tab切换区域
- 底部：对应Tab的内容区域

### 3. 个人中心页面 (`ProfileView.vue`)
**功能**：
- 用户信息展示
- 修改密码功能
- 修改Tushare Token功能
- 其他管理功能（如：关注股票管理、通知设置等）
- 退出登录

**UI设计**：
- 左侧：导航菜单（个人信息、修改密码、修改Token、其他设置）
- 右侧：对应功能的内容区域

### 4. 主菜单组件 (`StockMenu.vue`)
**位置**：左上角
**功能**：
- 显示"股票"菜单项
- 点击进入股票列表页面
- 可扩展其他菜单项（如：分析报告、监控提醒等）

### 5. 用户菜单组件 (`UserMenu.vue`)
**位置**：左下角（用户登录后显示）
**功能**：
- 显示用户头像/名称
- 下拉菜单包含：
  - 个人中心
  - 修改密码
  - 修改Token
  - 退出登录

## 交互流程

### 1. 用户登录流程
```
未登录 → 访问任何页面 → 重定向到登录页 → 登录成功 → 进入首页
```

### 2. 股票查看流程
```
首页 → 点击"股票"菜单 → 股票列表页 → 点击某股票 → 股票详情页
```

### 3. 个人中心流程
```
登录后 → 点击左下角用户菜单 → 选择"个人中心" → 进入个人中心页面
```

## API接口设计

### 1. 认证相关
- `POST /api/auth/login` - 用户登录
- `POST /api/auth/logout` - 用户退出
- `POST /api/auth/refresh` - 刷新Token

### 2. 用户管理
- `GET /api/user/profile` - 获取用户信息
- `PUT /api/user/password` - 修改密码
- `PUT /api/user/token` - 更新Tushare Token

### 3. 股票数据
- `GET /api/stocks` - 获取股票列表
- `GET /api/stocks/:code` - 获取股票详情
- `GET /api/stocks/:code/daily` - 获取日线数据
- `GET /api/stocks/:code/analysis` - 获取分析数据
- `POST /api/stocks/fetch` - 手动获取数据
